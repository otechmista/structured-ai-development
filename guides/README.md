# Coding Guides

Guias curtos para gerar e revisar codigo com IA.

Use com `04_CODE_VIEW.md`.

## Regra do projeto

- Caveman mode sempre.
- Curto.
- Direto.
- Sem filler.
- Sem repetir contexto.
- Explique so decisao, risco ou bloqueio.

## Uso

1. Complete as 4 Views.
2. Abra o guia da stack.
3. Copie regras uteis para `04_CODE_VIEW.md`.
4. Gere codigo.
5. Revise com `checklists/code-review.md`.

## Regras gerais

- Codigo simples.
- Nomes claros.
- Camadas separadas.
- Regras no dominio/application.
- Entrada validada na borda.
- Erros explicitos.
- Logs sem segredo.
- Testes para regra, erro e contrato.
- Dependencia nova so com motivo.
